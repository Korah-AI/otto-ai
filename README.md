# Description
<ul>
  <li>This is a very simple, completely free backend application to solve the issue of administrators blocking A.I. applications on devices.</li>
  <li>Users simply email their prompt Otto at "just.otto.it@gmail.com" and Otto responds within a minute. </li>
  <li>I came up with this idea as a joke at school during a physics class. Enjoy.</li>
</ul>

# Setup
<ul>
  <li>Paste the code in Google App Scripts</li>
  <li>Provide your own email and OpenAI API Key</li> 
  <li>Create a trigger for the "processEmails" function to occur every minute</li>
  <li>**Optional**</li>
    <li>Deploy via Google Cloud.</li>
</ul>

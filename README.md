# Description
<ul>
  <li>This is a very simple, completely free backend application to solve the issue of administrators blocking A.I. applications on devices.</li>
  <li>Users simply email their prompt Otto at [just.otto.it@gmail.com](url) and it responds within a minute. </li>
  <li>I came up with this idea as a joke at school during a physics class. Enjoy.</li>
</ul>

# Requirements
<ul>
  <li>Paste the code in Google App Scripts, provide your own email and OpenAI API Key, and create a trigger for the "processEmails" function to occur every minute. </li>
</ul>
